# Collector Chemistry OpenSea Policy

Collector Chemistry uses OpenSea only for read-only enrichment.

Allowed:

- collection metadata
- NFT metadata
- wallet NFT holdings
- activity history
- contextual stats when clearly labeled

Not allowed:

- buying
- selling
- swaps
- minting
- fulfillment
- signing
- transaction preparation
- private keys
- wallet custody

Collector Chemistry is not a marketplace or trading assistant.
