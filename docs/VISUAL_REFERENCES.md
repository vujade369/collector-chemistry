# Visual References — Collector Chemistry

## Profile Page Direction: Neon Pulse

Reference image:
`docs/references/profile-neon-pulse.png`

This is the current preferred direction for the individual wallet profile page.

## What to borrow

- Dark, compact, high-contrast page structure
- Magenta/pink accent system
- Glowing card borders used sparingly
- Stronger stat hierarchy
- Collector identity at the top
- Rounded card modules
- Taste map as a visual centerpiece
- Core signals displayed as compact collectible cards
- Top collections shown as clean ranked rows
- Compare CTA as a strong final action

## What not to copy too literally

- Do not make every card overly neon
- Do not add heavy animation
- Do not use fake data
- Do not turn the page into a generic dashboard
- Do not reduce the interpretive copy too much

## Product feeling

The page should feel like:
- a collector profile card
- a taste readout
- slightly game-like
- personal and revealing
- fun without becoming noisy

It should not feel like:
- a finance dashboard
- an NFT trading terminal
- a generic analytics page

## Layout order

1. Collector header
2. Stats row
3. Archetype / interpretation
4. Taste map
5. Core signals
6. Top collections
7. Compare CTA

## Core signals shown

- Top Collection
- Most Recent Acquisition
- First Mint
- Return Pattern
- Highest Bid
- Collecting Style

## Accent system

Primary accent:
- magenta / pink

Secondary accent:
- purple

Optional secondary signal accent:
- cyan / blue, used lightly

Use accents to guide attention, not decorate everything.
