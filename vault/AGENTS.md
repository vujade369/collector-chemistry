# AGENTS.md

## Purpose

This vault supports the Constellate product system.

Agents should use it to understand product direction, working rules, prompts, decisions, and feature plans.

## Product frame

Constellate helps people see the pattern in what they keep — and where it lights up with someone else’s.

It starts with NFT wallets but can expand into tickets, memberships, places, events, purchases, and other cultural artifacts.

The product is about recognition, not ranking.

## Core thesis

What you keep becomes a pattern.  
What overlaps becomes recognition.

## Working behavior

Before suggesting implementation:

1. Define the product question.
2. Identify the current source of truth.
3. Scope the change.
4. Protect existing behavior.
5. Make the smallest useful change.

## Codex behavior

Codex should:
- inspect before editing
- keep changes scoped
- return files changed
- return verification output
- avoid broad refactors
- avoid product-language invention unless asked
